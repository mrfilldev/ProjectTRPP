# ProjectTRPP
Учебный Web-проект.

Web-проект, который осуществляет свою функциональность с помощью браузеров (Google Chrome, Yandex Browser, Safari).

Продукт поддерживается с помощью виртуальной среды Virtualenv, реализован в большей части на языке Python. Проект открывается по адресу локальной сети.

#Техническое задание:
- отображение списка друзей
- отображение списка мороженого
- предоставление возможности выбора друга
- предоставление возможности выбора мороженого
- по итогу выбора друга и мороженого, и нажатия кнопки "угостить" предоставить информацию о погоде в городе друга, а так же вывести дружественную фразу о рекомендауии потребления мороженого
